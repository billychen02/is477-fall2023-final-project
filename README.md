# Analaysis of UCI Diagnostic Breast Cancer Data

## **Overview:**
The purpose of this repository is to reproduce the results reported of the "Breast Cancer Wisconsin (Diagnostic)" by Wolberg, W., Mangasarian, O., Street, N., and Street, W. (1995), utilizing UC Irvine Machine Learning Repository. The data are computed from a digitized image of a fine needle aspirate (FNA) of a breast mass, indicating the traits and characteristics of the nuclei cell captured in the image.
